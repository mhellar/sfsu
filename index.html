Venus of the Anthropocene

I work with the artist Lynn Hershman and wrote the software for her recent work, Venus of the Anthropocene. The work is an edition of 3 and was exhibited in New York, London and San Francisco in 2017. The facial recognition software that is capable of sentiment analysis and was written in Openframeworks, NodeJS and uses the SkyBiometry API

Sensor Driven Augmented Reality

This was a project for the Intel/Amazon Internet of Things hackathon in 2016. We created a lasercut house with emebedded sensors and actuators. Data from the sensors were streamed to an augemented reality app via the Amazon cloud. The Project won grand Prize.
